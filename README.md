
## NESTJS BLOG REST API
<p>It's my blog rest API made in Nestjs. I wrote this app to befriend with Nest</p><br/>
<p>Frontend part can be found here: https://github.com/NowackiPatryk/VUE_BLOG_FRONTEND</p>

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
